# QR code + unsplash generate

 說明 : 練習react + nodejs + mogode的串接

## 安裝方式


分別到front-end 及 back-end的資料夾安裝各package modules
### `cd back-end/`
### `npm install`

### `cd back-end/`
### `npm install`

## 使用方式

安裝完各資front-end 及 back-end的ppackage modules後，再分別啟動各資料夾的專案
### `cd back-end/`
### `npm run start`

### `cd back-end/`
### `npm run start`
